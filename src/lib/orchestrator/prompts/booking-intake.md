# Booking Intake — Чек-лист сбора данных

> Этот файл описывает, какие данные нужно собрать у клиента для сборки приложения записи на услуги.
> Используется LLM-агентом при парсинге брифа и формой intake на фронтенде.

---

## Обязательные поля

### Бизнес
| Поле | Тип | Пример | Описание |
|------|-----|--------|----------|
| `name` | string | "Blade & Fade" | Название заведения |
| `slug` | string | "blade-and-fade" | URL-ключ |
| `locale` | "ru" \| "en" | "ru" | Язык интерфейса |
| `currency` | string | "RUB" | Валюта |

### Услуги (минимум 1)
| Поле | Тип | Обязательное | Описание |
|------|-----|:---:|----------|
| `name` | string | ✅ | Название услуги |
| `price` | number | ✅ | Цена |
| `duration` | number | ❌ | Длительность в минутах |
| `category` | string | ❌ | Категория ("Стрижки", "Окрашивание") |
| `description` | string | ❌ | Описание услуги |
| `image` | file/url | ❌ | Фото |

---

## Опциональные поля

### Сотрудники / мастера
| Поле | Тип | Описание |
|------|-----|----------|
| `name` | string | Имя мастера |
| `bio` | string | Краткое описание (специализация, опыт) |
| `image` | file/url | Фото |

### Рабочие часы
| Поле | Тип | Описание |
|------|-----|----------|
| `day_of_week` | 0-6 | 0 = воскресенье, 1 = понедельник, … 6 = суббота |
| `start_time` | "HH:MM" | Время открытия |
| `end_time` | "HH:MM" | Время закрытия |
| `is_day_off` | boolean | Выходной день |

**Упрощённый формат от клиента**: "Пн-Пт 9:00-20:00, Сб 10:00-18:00, Вс — выходной"

### Брендинг
| Поле | Тип | По умолчанию | Описание |
|------|-----|:---:|----------|
| `primaryColor` | hex | #1a1a2e | Основной цвет |
| `secondaryColor` | hex | #5856D6 | Акцентный цвет |
| `subtitle` | string | auto | Слоган |

### Контакты
| Поле | Тип | Описание |
|------|-----|----------|
| `phone` | string | Телефон |
| `address` | string | Адрес заведения |

### Маркетинг
| Поле | Тип | Описание |
|------|-----|----------|
| `features` | array | 4 преимущества |
| `testimonials` | array | Отзывы клиентов |

---

## Автозаполнение (LLM генерирует если не указано)

| Что | Откуда | Пример |
|-----|--------|--------|
| `slug` | из `name` | "Blade & Fade" → "blade-and-fade" |
| `subtitle` | из ниши | "Запишись за 30 секунд. Без звонков." |
| `features` | из типа | ["Быстрая запись", "Напоминания", "Безопасная оплата", "Бесплатная отмена"] |
| `testimonials` | генерация | 3 отзыва от клиентов салона |
| `duration` | из типа услуги | "Стрижка" → 45 мин, "Окрашивание" → 120 мин |
| `workingHours` | стандартные | Пн-Пт 9-20, Сб 10-18, Вс выходной |
| `staff.bio` | из имени + услуг | "Старший мастер. Специализация: мужские стрижки." |

---

## Парсинг рабочих часов

LLM должен уметь распознавать разные форматы:

```
"Пн-Пт 9:00-20:00, Сб 10:00-18:00, Вс выходной"
→ [
    { day_of_week: 1, start_time: "09:00", end_time: "20:00", is_day_off: false },
    { day_of_week: 2, start_time: "09:00", end_time: "20:00", is_day_off: false },
    { day_of_week: 3, start_time: "09:00", end_time: "20:00", is_day_off: false },
    { day_of_week: 4, start_time: "09:00", end_time: "20:00", is_day_off: false },
    { day_of_week: 5, start_time: "09:00", end_time: "20:00", is_day_off: false },
    { day_of_week: 6, start_time: "10:00", end_time: "18:00", is_day_off: false },
    { day_of_week: 0, start_time: "00:00", end_time: "00:00", is_day_off: true },
  ]
```

---

## Валидация на входе

- `name` не пустой
- Минимум 1 услуга с `name` и `price >= 0`
- `duration` > 0 если указано
- Рабочие часы: `start_time < end_time` (если не выходной)
- Фото — jpg/png/webp, до 5MB

## Пример минимального брифа

```
Барбершоп "Blade & Fade".
Язык русский, рубли.

Услуги:
- Мужская стрижка, 1500₽, 45 мин
- Стрижка + борода, 2200₽, 60 мин
- Окрашивание, 3500₽, 90 мин

Работаем Пн-Сб 10:00-21:00, Вс выходной.
Адрес: ул. Пушкина 10.
```

## Пример полного брифа

```
Барбершоп "Blade & Fade".
Язык русский, рубли.
Цвет: тёмный (#1a1a2e).

Мастера:
- Алексей (стрижки, 5 лет опыта) [фото]
- Дмитрий (борода, 3 года) [фото]

Услуги:
- Мужская стрижка, 1500₽, 45 мин, категория "Стрижки"
- Стрижка + борода, 2200₽, 60 мин, "Стрижки"
- Моделирование бороды, 800₽, 30 мин, "Борода"
- Окрашивание, 3500₽, 90 мин, "Окрашивание"
- Камуфляж седины, 2000₽, 45 мин, "Окрашивание"

Работаем: Пн-Пт 10:00-21:00, Сб 10:00-19:00, Вс выходной.
Адрес: ул. Пушкина 10, м. Чистые пруды.
Тел: +7 999 123-45-67.

Отзыв: "Хожу полгода, лучший барбершоп в районе" — Сергей.
```

# Ecommerce Intake — Чек-лист сбора данных

> Этот файл описывает, какие данные нужно собрать у клиента для сборки интернет-магазина в Telegram Mini App.
> Используется LLM-агентом при парсинге брифа и формой intake на фронтенде.

---

## Обязательные поля

### Бизнес
| Поле | Тип | Пример | Описание |
|------|-----|--------|----------|
| `name` | string | "StyleHub" | Название магазина |
| `slug` | string | "stylehub" | URL-ключ (генерируется из name, lowercase, дефисы) |
| `locale` | "ru" \| "en" | "ru" | Язык интерфейса |
| `currency` | string | "RUB" | Валюта (ISO код) |

### Товары (минимум 1)
| Поле | Тип | Обязательное | Описание |
|------|-----|:---:|----------|
| `name` | string | ✅ | Название товара |
| `price` | number | ✅ | Цена в выбранной валюте |
| `category` | string | ❌ | Категория (группировка в каталоге) |
| `description` | string | ❌ | Описание (1-2 предложения) |
| `image` | file/url | ❌ | Фото товара |
| `stock_quantity` | number | ❌ | -1 = безлимит, 0 = нет в наличии, >0 = ограничено |

**Источники данных**: CSV/Excel файл, текстовый список, фото + подписи.

---

## Опциональные поля

### Брендинг
| Поле | Тип | По умолчанию | Описание |
|------|-----|:---:|----------|
| `primaryColor` | hex | #007AFF | Основной цвет бренда |
| `secondaryColor` | hex | #5856D6 | Акцентный цвет |
| `logo` | file/url | — | Логотип |
| `heroImage` | file/url | — | Фоновое изображение для баннера |
| `subtitle` | string | auto | Слоган / подзаголовок |

### Маркетинг
| Поле | Тип | Описание |
|------|-----|----------|
| `features` | array<{icon, title, description}> | 4 преимущества ("Быстрая доставка", "Свежие продукты") |
| `testimonials` | array<{name, text, rating}> | Отзывы клиентов (если нет — генерируем 3 шт.) |
| `promoText` | string | Текст промо-баннера ("Бесплатная доставка от 2000₽") |
| `promoCode` | string | Промокод для скидки |

### Контакты и доставка
| Поле | Тип | Описание |
|------|-----|----------|
| `phone` | string | Телефон для связи |
| `address` | string | Адрес (если есть самовывоз) |
| `deliveryInfo` | string | Условия доставки |
| `workingHours` | string | Часы работы |

---

## Автозаполнение (LLM генерирует если не указано)

| Что | Откуда | Пример |
|-----|--------|--------|
| `slug` | из `name` | "Urban Style" → "urban-style" |
| `subtitle` | из ниши + товаров | "Стильная одежда с доставкой" |
| `features` | из типа бизнеса | 4 стандартных преимущества для fashion |
| `testimonials` | генерация | 3 правдоподобных отзыва |
| `categories` | из товаров | Уникальные category из списка товаров |
| `promoText` | из контекста | "Бесплатная доставка при заказе от 3000₽" |
| FAQ | из ниши | 4 типовых вопроса |

---

## Валидация на входе

- `name` не пустой
- Минимум 1 товар с `name` и `price >= 0`
- `currency` — валидный ISO код
- `locale` — "ru" или "en"
- Если есть фото — формат jpg/png/webp, до 5MB каждый
